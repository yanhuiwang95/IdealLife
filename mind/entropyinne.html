从熵增到负熵

我们终将老去，但可以有序地老去。

作者: yanhui 发布日期: 2025-07-29
什么是熵增？

在热力学中，熵（entropy）代表的是一个系统的无序程度。第二热力学定律指出，封闭系统的熵只会不断增加，也就是说，任何孤立系统终将走向混乱、衰败与不可逆的无序状态。

一个房间如果不整理，最终会变得杂乱不堪。这就是熵增。

而人体和人生，其实也是一个开放系统，虽然可以短期维持秩序，但如果不主动对抗，就会不可避免地受到熵增的影响。

  • 身体层面：随着年龄增长，新陈代谢变慢、肌肉流失、认知下降、免疫功能减弱，这些都体现了生物系统走向熵增。
  • 心理层面：目标感丧失、情绪波动、社交退化、自我价值感降低，甚至空虚与焦虑。
  • 生活层面：日常节律混乱、行动力下降、注意力涣散、生活空间变得杂乱无章。

退休后，原本由职业和社会角色维持的“秩序感”突然瓦解，是人生熵增的一个高风险时点。

什么是负熵？

薛定谔在其名著《生命是什么？》中提出：生命以吸收“负熵”维持其有序状态。换言之，生命系统之所以能逆转自然趋势、维持复杂结构和功能，是因为持续吸收和利用外界的信息、能量和秩序。

简单说：负熵，就是对抗混乱、构建秩序的能力。

《生命是什么？》 - 概要

《生命是什么？》是诺贝尔奖获得者、物理学家埃尔温·薛定谔（Erwin Schrödinger）于1944年出版的一本具有里程碑意义的科普著作。该书基于他在都柏林圣三一学院的系列公开讲座，旨在从物理学的角度探讨生命的本质和遗传信息的物理基础。

书中核心的观点包括：

  1. “非周期性晶体”概念： 薛定谔提出，遗传信息可能储存在一种“非周期性晶体”中，这种结构既能稳定地承载大量信息，又不像普通晶体那样重复排列，从而为生命复杂性的编码提供了可能。这一理论极大地启发了詹姆斯·沃森（James Watson）和弗朗西斯·克里克（Francis Crick）在1953年发现DNA双螺旋结构。
  2. “负熵”概念： 他探讨了生命体如何在看似违反热力学第二定律（即熵增原理，系统趋向于无序）的情况下，维持其高度有序的状态。薛定谔认为，生命体通过“以负熵为生”（feeding on negative entropy），即不断从环境中获取能量和有序物质，来对抗自身的衰变和无序化。
  3. 物理学与生物学的融合： 该书是物理学家对生物学基本问题的深刻思考，促进了物理学、化学和生物学之间的交叉研究，为分子生物学和生物物理学的发展奠定了基础。

简而言之，这本书以其跨学科的视角和富有洞察力的思考，启发了科学界对遗传物质物理本质的探索，并对理解生命体的秩序和信息储存方式产生了深远影响。

对抗生理熵增 - 保持健康

也就是维持身体系统的有序性，这是所有负熵机制的物理基础。

规律的作息、科学的锻炼、合理的饮食、足够的恢复，本质上就是在为身体提供“秩序输入”。

不管喜不喜欢，健康的生活习惯是维持系统稳定的最低要求，是所有自由的前提。

对应负熵机制：
持续稳定的输入 → 对抗生理熵增（老化、退化）→ 系统结构维持 → 自主性增强

对抗心理熵增 - 追求热爱

热爱是注意力与能量的集中器，是大脑认知秩序的源泉，因此使维持心理系统的有序性的最佳选择。

做自己真正感兴趣的事，不是逃避，而是一种高效的能量循环。

当注意力集中、情绪积极，大脑会释放奖励机制，内在秩序增强。

对应负熵机制：
持续的心理投入 → 神经可塑性与认知结构强化 → 内在秩序构建 → 成就感与动力维持

对抗外部秩序熵增 - 创造价值

人是社会性动物，孤立等于熵死。

通过社交、志愿服务、分享、创造内容等方式，将自己与外界连接起来，不只是被动接受秩序，而是主动输出秩序。

“连接”本身就是一个负熵行为：信息交换越频繁，系统越稳定。

对应负熵机制：
输出价值 → 建立反馈回路 → 增强意义感 → 情绪稳定，认知有序

退休生活的“反熵系统”，不是热情，而是架构 - 从热力学到系统论，从行为管理到能量流通，从外部刺激到内部回路

许多人以为，理想的退休生活依赖于“自律”与“规划”。但真正持久有效的，并不是意志力的堆砌，而是一个有意设计的低熵结构。

所谓简单性原则、80/20原则与系统原则，并不只是生活建议，而是负熵输入的系统架构：从结构稳定性、能量配置效率、信息流动路径三个核心维度，为我们的生活系统不断引入秩序，构建出对抗熵增的机制性秩序网络。

这套负熵框架，不靠激情驱动，而靠结构自稳；不是偶发的奋起，而是日复一日的可持续有序。

简单原则 = 降低系统复杂度，减少熵源

为什么复杂性等于熵呢？系统越复杂，可选项越多，路径越长，状态越多，维持秩序所需能量越高。所以，冗余结构=潜在熵源。

负熵机制：压缩决策路径，降低系统复杂性，防止熵扩散。

“简单”并不意味着少做事，而是有意地进行“信息熵压缩”：

  • 明确生活的核心任务（健康、热爱、价值）
  • 排除非必要干扰（外界期待、无效比较、日程堆砌）
  • 降低行为路径分岔（固定节律、减少“今天做不做”的决策负担）

结果：生活能量更集中；精力配置更有序。

每天行为重复性高 → 系统自组织 → 引入结构性负熵

对应生物系统例子：像细胞通过简化代谢路径、限制基因表达量波动来节省能量，这就是生理层面的“简单原则”。

80/20原则 = 优化能量通道，引入关键秩序点

在任何系统中，80%的秩序通常由20%的结构承担。通过识别“秩序杠杆点”，可以以最小投入对抗最大熵增。

负熵机制：在最低能耗路径上输入最大秩序，构建高杠杆点。

  • 健康：只做“复利型行为”（如睡眠、力量训练），胜过堆时间
  • 热爱：持续少数深度活动（如写作、创作、获取技能），远胜于碎片化尝试
  • 价值：专注高反馈的连接点（如长期志愿活动、稳定社群输出），胜过无目的社交

结果：减少能量耗散在“无效高熵边缘区”（如信息过载、情绪消耗型任务）

构建“负熵回路”：关键活动自动带来反馈与秩序（健康 → 活力 → 投入 → 成就 → 情绪稳定 → 再投入）

对应生态系统例子：生态网络中，关键节点（如传粉者或顶级捕食者）决定整个系统稳定性，这正是自然界中的“80/20式负熵”。

系统原则 = 建立自动化秩序回路，引入自组织能力

最大的问题不是一次做得好，而是能否持续产生秩序。人类的意志力波动很大，仅靠意愿驱动容易熵散。

负熵机制：生活行为内嵌进机制，形成自组织反馈，持续生成秩序。

系统原则的核心就是，用结构，代替意志；用机制，代替反复选择。

  • 固定信息流动路径 - 不是“今天想不想锻炼”，而是“周跑三次，力量两次”。
  • 降低随机性决策的能量消耗 - 不是“今天做什么”，而是“今天30分钟吉他”。
  • 通过行为制度化引入高频小型负熵输入点 - 不是“感觉好才去”，而是“周六群跑”。

结果：一旦系统建立，它将自动回收混乱、排除冗余、生成结构，像有机体一样自我维护。

对应计算系统例子：好的操作系统不是靠人工不断调试，而是靠稳定的后台进程、规则触发器、异常修复机制维持秩序。

退休不是躺平，而是对抗熵增的精致工程。我们终将老去，但可以有序地老去。
